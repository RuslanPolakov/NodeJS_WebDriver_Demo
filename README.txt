This project is demo test of calculator website (https://web2.0calc.ru/) using  NodeJS, Selenium WebDriver and Mocha

Project dependencies (already in node_modules) :
WebDriver (could be obtained by 'npm install selenium-webdriver')
Mocha (JavaScript test framework, could be obtained by 'npm install mocha')

chromedriver.exe (could be downloaded from - https://sites.google.com/a/chromium.org/chromedriver/)
CalculatorPage.js - contains calculator page mapping
Tests.js - contains selenium test

To execute project should be used command 'mocha Tests.js'
